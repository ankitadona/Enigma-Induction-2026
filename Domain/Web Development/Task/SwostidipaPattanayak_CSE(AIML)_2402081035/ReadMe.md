# 🌐 Enigma Club Induction Task – Website Project

## 📌 Overview

This project was developed as part of the **Enigma Club Induction Task**. The goal was to create a structured and visually engaging website for the club while demonstrating core frontend development skills and deployment knowledge.

---

## 🚀 Features

* Multi-page website with dedicated sections
* Clean and simple user interface
* Smooth navigation across pages
* Basic interactivity using JavaScript
* Organized and maintainable code structure
* Responsive design basics

---

## 🛠️ Tech Stack

* **HTML5** – Website structure
* **CSS3** – Styling and layout
* **JavaScript** – Interactivity

---

## 📂 Project Structure

```id="k3m92x"
├── index.html        # Homepage  
├── team.html         # Team members section  
├── projects.html     # Projects showcase  
├── blogs.html        # Blog section  
├── contact.html      # Contact information  
├── style.css         # Styling file  
├── script.js         # JavaScript functionality  
```
